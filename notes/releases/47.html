DrScheme Release Notes

Current Release: 47
Release Date: January 11, 1997.

Send bug reports to: plt-bugs@cs.rice.edu

The following is a summary of the major changes in this release from the previous one.

DrScheme

  • The entire MrEd graphical toolbox is now available for development within DrScheme.

Analysis

  • The Analysis tool (MrSpidey) is temporarily unavailable in release 47 and will reappear in a future release.

Donkey

  • The Stepper tool (Donkey) is temporarily unavailable in release 47 and will reappear in a future release.

MrEd

  • See plt/mred/HISTORY and plt/mred/system/history in the distribution for changes.

MzScheme

  • See plt/mzscheme/HISTORY in the distribution for changes.

Known Problems

  • See plt/openbugs in the distribution.
