PLT Release Notes

Current Release: 51
Release Date: March 25, 1998.

The following is a summary of the major changes in this release of the PLT software from the previous one.

New Programs

  • DrScheme Jr is a new text-only version of DrScheme. It requires less resources, but provides only a read-eval-print loop. DrScheme Jr does not provide a built-in program editor or a graphical toolbox.
  • mzc is a new compiler that transforms MzScheme programs to native code libraries. mzc is not yet an optimizing compiler; the performance of its output is comparable running the Scheme source directly in MzScheme.

MrSpidey

MrSpidey is now distributed as a separate tool. When installed, it may be invoked from DrScheme.

Networking facilities

The networking-related routines have now been incorporated into the distribution. They are in the collection `net', and consist of the packages
  • cgi provides routines for writing CGI scripts
  • nntp communicates with a Usenet server
  • pop3 communicates with a POP-3 server
  • url parses and establishes HTTP connections
  • mail sends email (preliminary, currently only works under Unix)
Documentation is in a subdirectory `docs', of the net collection directory. That subdirectory contains files named <package>.txt, where <package> ranges over the above names.

Miscellany

Console I/O has been sped up, and is interruptible by using the Break button.

See the HISTORY files in notes/drscheme, notes/mzscheme, and notes/mred for descriptions of other bug fixes and minor changes.

The Future

IMPORTANT: Expect major changes in the MrEd graphical toolbox with version 53 or so. Likely changes include:
  • removing the useless size arguments for initializing mred:frame%, mred:button%, etc. instances;
  • changing the "nothing" value (e.g., no parent frame) from null to #f; and
  • replacing bitwise-ored wx:const-XXX numerical constants with lists of 'wx:const-XXX symbols.
