DrScheme Release Notes

Current Release: 46
Release Date: December 5, 1996.

Send bug reports to: plt-bugs@cs.rice.edu

The following is a summary of the major changes in this release from the previous one.

DrScheme

  • All of MzScheme's syntax is now supported in DrScheme, except for units with signatures.

Analysis

  • The Analysis tool (MrSpidey) is temporarily unavailable in release 46 and will reappear in a future release.

Donkey

  • The Stepper tool (Donkey) is temporarily unavailable in release 46 and will reappear in 47.

MrEd

  • See plt/mred/HISTORY and plt/mred/system/history in the distribution for changes.

MzScheme

  • See plt/mzscheme/HISTORY in the distribution for changes.

Known Problems

  • DrScheme internally normalizes pathnames, resolving all links to absolute paths. This causes problems with systems that use automounting.
  • See also plt/openbugs in the distribution.
